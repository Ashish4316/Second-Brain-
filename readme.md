# secondBrain

A small Node + TypeScript project for personal notes and utilities.

## Setup

Install dependencies:

```bash
npm install
```

Build (if a `build` script exists) or compile with TypeScript:

```bash
npm run build
# or
npx tsc
```

Run (adjust depending on your scripts):

```bash
npm start
# or for development
npx ts-node src/index.ts
```

## Project structure

- `src/` — TypeScript source files
- `package.json` — NPM scripts & dependencies
- `tsconfig.json` — TypeScript config
- `.gitignore` — generated; ignores node_modules, build outputs, env files, editor dirs

## Notes

This repository was initialized locally. Customize scripts in `package.json` as needed.
